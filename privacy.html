Privacy Policy

Last Updated: January 31, 2026

Your privacy is important to us. This Privacy Policy explains what data the GLOBAL OFFICIAL Discord bot collects, how it is used, stored, and protected. By using the Bot, you consent to the data practices described in this policy.

1. Data We Collect

The Bot collects and stores the following types of data to provide its features and functionality:

1.1 Automatically Collected Data

  • Discord User IDs - To identify users and track their interactions
  • Discord Server (Guild) IDs - To provide server-specific configurations
  • Channel IDs - For channel-specific features and whitelisting
  • Role IDs - For moderation role configurations and role management features
  • Message IDs - For snipe/edit snipe features, quote system, and logging
  • Timestamps - For cooldowns, activity tracking, and "last seen" functionality

1.2 User-Generated Content

  • Message Content - Stored temporarily for snipe/edit snipe features (limited history)
  • Confessions - Anonymous confession messages with minimal metadata
  • Quotes - User-saved quotes with associated content and metadata
  • Marriage Data - User partnerships created through the marriage system
  • Suggestions - Suggestions submitted through the suggestion system
  • Custom Prefixes - Server-specific command prefixes
  • Bot Configuration - Settings for moderation, logging, automation, and features

1.3 Activity & Tracking Data

  • Server Activity - Message counts, user activity timestamps for analytics
  • Last Seen - Timestamps of when users were last active
  • Counting Game Progress - Current counts and user participation data
  • Voicemaster Data - Temporary voice channel ownership and configurations
  • Moderation History - Active mutes, role backups during mutes, moderation actions (if logging enabled)
  • Hall of Fame - Messages with high reaction counts

1.4 Server Backup Data

  • Server Structure - Channels, roles, permissions when backups are created
  • Server Settings - Server name, icon, and configuration
  • Member Roles - Role assignments for backup/restoration purposes

2. How We Use Your Data

Collected data is used strictly to:

  • Provide bot features, commands, and functionality
  • Store server and user preferences and configurations
  • Track activity for analytics and leaderboards
  • Enable moderation tools (mute tracking, role restoration)
  • Backup and restore server configurations
  • Improve bot performance, reliability, and user experience
  • Generate AI responses (when AI features are enabled in whitelisted channels)
‚úÖ We Never:
  • Sell your data to third parties
  • Share your data except as required to provide Bot services
  • Use your data for unrelated advertising or marketing

3. Data Storage & Security

Data is stored securely using the following methods:

  • SQLite Databases - Local encrypted databases (server_activity.db, server_backup.db, moderation.db, etc.)
  • JSON Files - Configuration files for settings and preferences
  • Access Control - Data is only accessible to Bot developers and authorized systems

While we implement industry-standard security measures, no system is 100% secure. Data is stored on secure private servers with restricted access.

4. Data Retention

Data retention varies by feature:

  • Permanent (until manual deletion): Server configurations, prefixes, role settings, backups
  • Until feature is disabled: Confessions, marriages, quotes, activity tracking
  • Temporary (limited history): Snipe data (last 50 messages), AI response cooldowns (30 minutes)
  • Automatic cleanup: Expired mutes, inactive voicemaster channels, old counting data

Unused or unnecessary data may be periodically purged to optimize database performance.

5. Third-Party Services & APIs

The Bot interacts with the following third-party services:

  • Groq AI - Provides AI-powered responses when AI features are enabled. Messages sent to Groq may be subject to their privacy policy.
  • Discord API - All bot interactions rely on Discord's platform and are subject to Discord's Privacy Policy.
  • Image/GIF APIs - External services for meme generation, image processing, and entertainment features.

These third-party services have their own terms and privacy policies. We recommend reviewing them if you use features that rely on these services.

6. User Rights & Data Control

You have the following rights regarding your data:

6.1 Data Access

You may request to see what data we have stored about you or your server.

6.2 Data Deletion

You may request deletion of your data by:

  • Removing the Bot from your server (server data will be retained for backups unless deletion is requested)
  • Contacting us through our Discord Server

6.3 Opt-Out

You can opt out of specific features:

  • AI features can be disabled by not whitelisting channels
  • Activity tracking can be disabled per server
  • Individual features can be disabled through bot commands

7. Children's Privacy

The Bot is not directed at children under the age of 13. We do not knowingly collect data from children under 13. Discord's Terms of Service require users to be 13 or older. If we become aware that we have collected data from a child under 13, we will take steps to delete such data.

8. Data Sharing & Disclosure

We do not sell, trade, or otherwise transfer your data to outside parties except:

  • To Provide Services: Data processed by third-party APIs (Groq AI, Discord) as necessary for bot functionality
  • Legal Requirements: If required by law, court order, or government regulation
  • Safety & Security: To protect against fraud, abuse, or security threats

9. International Data Transfers

Data may be stored and processed on servers located in various jurisdictions. By using the Bot, you consent to the transfer of your data to these locations.

Changes will be effective immediately upon posting. Continued use of the Bot after changes indicates acceptance of the updated policy. We encourage you to review this policy periodically.
